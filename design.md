# Bihar Rainfall Dashboard Design

## Design Philosophy
- **Visual Style**: Modern editorial design inspired by data journalism platforms
- **Color Palette**: Monsoon-inspired colors with deep teals, muted blues, and warm earth tones
- **Typography**: Clean sans-serif for readability with Hindi language support
- **Layout**: Grid-based with ample whitespace for clarity

## Color Scheme
- Primary: Deep teal (#1e4d4b) - representing monsoon waters
- Secondary: Warm earth (#8b6f47) - representing Bihar's agricultural landscape  
- Accent: Muted blue (#4a90a4) - for interactive elements
- Background: Soft cream (#faf9f7) - for readability
- Text: Charcoal (#2d3748) - for strong contrast

## Visual Effects
- Subtle parallax scrolling for hero section
- Animated data transitions using Anime.js
- Interactive heatmap with hover effects
- Smooth chart animations with ECharts.js
- Gradient overlays for visual depth

## Typography
- Display: Playfair Display (serif) for headings
- Body: Inter (sans-serif) for content
- Hindi text support with Noto Sans Devanagari
